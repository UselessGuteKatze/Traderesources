Для полноценного запуска проекта необходимо:

  • Настроить базу данных
    1. Поднять в Docker контейнер postgres: docker run --name {containerName} -p {port}:5432 -e POSTGRES_PASSWORD={password} -e POSTGRES_HOST_AUTH_METHOD=password -d postgis/postgis
    2. Пароль и порт должны совпадать с данными введеными при создании проекта
    3. Создать там бд (которую указывали при создание проекта)
    4. Проверить конфиги подключения к бд (appsettings.dbkeys.Development.json)
  • Настроить локальный SMTP сервер (почтовик)
    1. Поднять в Docker контейнер smtp сервера: docker run -p 3000:80 -p 2525:25 rnwood/smtp4dev:v3
    2. Проверить конфиги подключения к smtp серверу (appsettings.core.json (DefaultEmailSettings))
  • Настроить host файл
    1. Скопировать содержимое !Development\hosts.txt в ваш хост файл (в Windows это C:\Windows\System32\drivers\etc\hosts)
